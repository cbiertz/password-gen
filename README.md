# Random Password Generator

## Purpose
Built as an exersise to learn JavaScript, this application will generate a random password after prompting a user to choose their criteria. 

Criteria options are:
Length
Lowercase letters
Uppercase letters
Numbers
Special Characters

## Built With
* HTML
* CSS
* JavaScript
